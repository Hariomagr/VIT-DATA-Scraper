# VIT-DATA-Scraper
Scrapes all user information from vtopbeta

Simply download all the files and run the required python files as per your interest.
First enter your registration id and password and choose semester
The result will be saved in respective json file
For coursepage, run the file as python course_page.py and choose the number for the subject and sit back and relax till all the files are downloaded and organised in the same folder
